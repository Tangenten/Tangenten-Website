# GitHub Pages — Hello World

A simple "Hello World" [GitHub Pages](https://pages.github.com/) site built with raw HTML, CSS, and JavaScript.

## Quick Start

1. Clone or fork this repo.
2. Open `index.html` in your browser — or run a local server:
   ```bash
   python -m http.server 8000
   ```
3. Edit the files and push to GitHub to deploy.

## Deploy to GitHub Pages

1. Push to a GitHub repository.
2. Go to **Settings > Pages**.
3. Set the source branch to `main` (or `master`) and folder to `/` (root).
4. Your site is live at `https://<username>.github.io/<repo-name>/`.
